# JavaScript Pattern Generation Project

## Project Description

This project demonstrates how to generate a hollow diamond star (`*`) pattern using JavaScript. The implementation uses nested loops and string manipulation to correctly format spaces and stars to produce the desired output pattern.

## Features

* Uses JavaScript loops and conditional statements
* Demonstrates pattern generation logic
* Prints a hollow diamond pattern in the console
* Clean and readable implementation

## Technologies Used

* JavaScript
* Node.js (for running the script)

## Implementation

The program uses:

* A loop to generate the upper half of the pattern
* A loop to generate the lower half
* Conditional checks to print stars at the boundaries and spaces inside

## Example Output

```
    *
   * *
  *   *
 *     *
*       *
 *     *
  *   *
   * *
    *
```

## How to Run

1. Clone the repository
2. Navigate to the project director
3. Run the following command:

```
node pattern.js
```
